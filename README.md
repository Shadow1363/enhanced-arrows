# Water Arrow

A simple datapack that adds craftable special craftable arrows.

## Arrows

### Water
Spawns water

## To build

`zip -r WaterArrow.zip data pack.mcmeta`

## To get in creative with commands

`/give @a minecraft:arrow[minecraft:custom_data={water_arrow:1b}] 64`
